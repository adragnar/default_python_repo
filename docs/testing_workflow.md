You must run this testing workflow before executing any changes to the codebase.

1. Identify tests that you need to write to cover the new functionality you are adding.
2. Write the tests in advance. Save them in a new python file to the tests/ directory.
3. Make the changes to the codebase.
4. Run the tests using `uv run pytest`.
5. Fix any tests that fail.
6. Repeat steps 4 and 5 until all tests pass.
